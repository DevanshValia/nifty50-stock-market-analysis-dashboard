# 📈 NIFTY 50 Stock Market Analysis Dashboard

An end-to-end **Financial Analytics Dashboard** built using **PostgreSQL, SQL, Power BI, and DAX** to analyze historical performance of NIFTY 50 stocks.

This project demonstrates advanced SQL, data modeling, DAX calculations, and interactive dashboard design for financial data analysis.

# Project Objectives

- Analyze historical performance of NIFTY 50 stocks.
- Calculate CAGR, Volatility, Daily Returns and Turnover.
- Build interactive Power BI dashboards.
- Compare two stocks using Risk vs Return analysis.
- Generate an automatic investment verdict.
